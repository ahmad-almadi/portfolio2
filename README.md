# # Ahmad Portfolio

A modern, high-performance portfolio built with Next.js and React, designed to showcase my work, skills, and projects through a clean user interface and smooth user experience.

## Overview

This portfolio is an interactive web application that reflects my approach to full-stack development, combining performance, design, and intelligent features in a single platform.

## Tech Stack

* Next.js – Server-side rendering and routing
* React – Component-based architecture
* TypeScript – Type safety and maintainability
* CSS / Tailwind – Responsive and modern styling
* Next.js API Routes – Backend functionality
* Gemini AI – Integrated chatbot assistant
* Resend API – Email handling for contact form

## Features

* Modern and responsive design
* AI-powered chatbot assistant
* Contact form with real-time email delivery
* Optimized performance and fast loading
* Conversation memory handling for chat
* Cross-device compatibility

## Project Structure

* /app – Application routes using Next.js App Router
* /components – Reusable UI components
* /api – Backend endpoints (chat and contact)
* /lib – Utilities and configurations

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### Install dependencies

```bash
npm install
```

### Setup environment variables

Create a `.env.local` file and add:

```env
GEMINI_API_KEY=your_key
RESEND_API_KEY=your_key
CONTACT_TO_EMAIL=your_email
```

### Run the project

```bash
npm run dev
```
